Telecom Customer Churn Analysis (SQL | Power BI | Python ML)
•	Built an end-to-end ETL pipeline by importing telecom data into MySQL, cleaning null values, transforming tables, and creating analytical views for reporting.
•	Conducted EDA in Power BI, creating DAX measures, calculated columns, and KPIs to analyze churn patterns, customer segments, tenure groups, and revenue trends.
•	Developed a Random Forest churn prediction model (~86% accuracy) in Python using Jupyter Notebook and integrated the prediction outputs back into Power BI dashboards for actionable retention insights.
